NC-News-API

Excelent CRUD application!

Available endpoints

  • GET /api/topics

    Retrieve a list of all topics.

    Example response:

      
{
  "topics": [

    },
    {
      "slug": "cooking",
      "description": "Hey good looking, what you got cooking?"
    }
  ]
}
